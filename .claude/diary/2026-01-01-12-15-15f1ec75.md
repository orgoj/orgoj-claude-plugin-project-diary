## Lessons Learned

### Technical Lessons:
- Plugin `commands/` = distributed with plugin, available to users
- Project `.claude/commands/` = local dev workflow, not distributed
- `superpowers-developing-for-claude-code:working-with-claude-code` skill has reference docs for slash commands

### Process Lessons:
- Read skill descriptions carefully before invoking - match to actual task
- When creating dev tools, consider if they belong in plugin or project
- README updates must cover feature changes, not just version numbers

### To Remember for CLAUDE.md:
